O Quadro de horários é um projeto desenvolvido para organizar e exibir os horários das aulas de forma prática e intuitiva. Ele permite que os estudantes visualizem suas disciplinas distribuídas por dias da semana e períodos, facilitando o planejamento da rotina acadêmica e o acompanhamento das turmas. O sistema oferece uma forma prática de consultar os horários das aulas, permitindo que os estudantes acompanhem suas disciplinas e ajustem a rotina de acordo com o semestre. Com uma interface simples e de fácil navegação, o projeto possibilita criar, visualizar e atualizar horários conforme necessário, ajudando os alunos a manterem seus compromissos acadêmicos organizados e a otimizarem o tempo de estudo. O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, garantindo uma aplicação responsiva, interativa e de fácil manutenção.
